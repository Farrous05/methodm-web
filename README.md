## How to Edit

All content is in the HTML files ‚Äî no build step, no framework. Open any `.html` file and edit directly.

- **Copy / text** ‚Üí edit inside the HTML files
- **Styles** ‚Üí `css/style.css`
- **Interactions** ‚Üí `js/main.js`

CSS variables are defined at the top of `style.css` under `:root {}`.
